# 5.1 다채로운 웹 서버

웹 서버는 매일 수십억 개의 웹페이지를 쏟아 낸다.  
웹 서버는 HTTP 요청을 처리하고 응답을 제공한다.  
**'웹 서버'** 라는 용어는 웹 서버 소프트웨어와, 웹페이지 제공에 특화된 장비(컴퓨터와 같은) 양쪽 모두를 가리킨다.

웹 서버는 기능, 형태, 크기가 다양하다. 열 줄짜리 조그마한 펄 스크립트 웹 서버가 있는가 하면, 50메가바이트짜리 안전한 상용 엔진도 있고, 아주 작은 기판 위의 서버도 있다.  
그러나 기능은 달라도, **모든 웹 서버는 리소스에 대한 HTTP 요청을 받아서 콘텐츠를 클라이언트에게 돌려준다.**

---

## 5.1.1 웹 서버 구현

웹 서버는 HTTP 및 그와 관련된 TCP 처리를 구현한 것이다.  
웹 서버는 자신이 제공하는 리소스를 관리하고, 웹 서버를 설정·통제·확장하기 위한 관리 기능을 제공한다.

웹 서버가 하는 일:
- HTTP 프로토콜을 구현한다.
- 웹 리소스를 관리한다.
- 웹 서버 관리 기능을 제공한다.

웹 서버는 **TCP 커넥션 관리에 대한 책임을 운영체제와 나눠 갖는다.**  
운영체제는 하드웨어를 관리하고 TCP/IP 네트워크 지원, 웹 리소스를 유지하기 위한 파일 시스템, 현재 연산 활동을 제어하기 위한 프로세스 관리를 제공한다.

> **Java 개발자 관점**: 우리가 Spring Boot를 띄울 때 내장 톰캣(Tomcat)이 바로 이 '웹 서버 소프트웨어'에 해당한다.  
> 톰캣은 HTTP 프로토콜 파싱과 커넥션 관리를 담당하고, 실제 소켓의 생성/해제나 TCP/IP 스택은 운영체제(JVM을 거쳐)에 위임한다. 즉 "TCP 관리 책임을 OS와 나눠 갖는다"는 말이 그대로 적용된다.

웹 서버는 여러 가지 형태가 가능하다.
- 다목적 소프트웨어 웹 서버를 표준 컴퓨터 시스템에 설치하고 실행할 수 있다.
- 어떤 회사들은 사용자에게 판매하는 전자기기 안에 몇 개의 컴퓨터 칩만으로 구현된 웹 서버를 내장시켜서 완전한 관리 콘솔로 제공한다.

## 5.1.2 다목적 소프트웨어 웹 서버

다목적 소프트웨어 웹 서버는 네트워크에 연결된 표준 컴퓨터 시스템에서 동작한다.  
아파치(Apache)나 W3C의 직소(Jigsaw) 같은 오픈 소스 소프트웨어를 사용할 수도 있고, 마이크로소프트나 아이플래닛의 웹 서버 같은 상용 소프트웨어를 사용할 수도 있다.  
웹 서버 소프트웨어는 거의 모든 컴퓨터와 운영체제에서 동작한다.

수만 가지 종류의 웹 서버 프로그램이 존재하지만(직접 만들어진 것과 특수 목적용 웹 서버를 포함해서), 그중 몇 가지 웹 서버 소프트웨어만이 널리 사용된다.

### 넷크래프트 조사 (2014년 8월 기준)

| 웹 서버 | 시장 점유율 | 비고 |
|---------|------------|------|
| 마이크로소프트 | 약 37% | 점유율 1위 |
| 아파치(Apache) | 약 35% | 근소한 차이로 2위로 밀려남 |
| nginx | 약 14% | 최근 수년간 꾸준히 증가 |

> 한때 아파치가 압도적 1위였으나, 조사 시점(2014년) 기준으로는 마이크로소프트에 근소하게 밀려 2위로 내려간 상황이다. nginx는 가벼운 이벤트 기반 아키텍처를 무기로 빠르게 점유율을 늘렸다.

## 5.1.3 임베디드 웹 서버

**임베디드 웹 서버(Embedded Web Server)** 는 일반 소비자용 제품(예: 프린터나 가전제품)에 내장될 목적으로 만들어진 작은 웹 서버이다.  
임베디드 웹 서버는 사용자가 그들의 일반 소비자용 기기를 간편한 웹 브라우저 인터페이스로 관리할 수 있게 해준다.

몇몇 임베디드 웹 서버는 **1제곱인치도 안 되는 크기**로 구현되어 있고, 보통 최소한의 기능만을 제공한다.

아주 작은 임베디드 웹 서버의 예:
- 아이픽(IPic) 성냥 머리 크기 웹 서버
- 넷미디어 사이트플레이어(SitePlayer) SP1 이더넷 웹 서버

> **Java 개발자 관점**: 라우터 설정 페이지(`192.168.0.1`)에 접속할 때 마주하는 그 관리 화면이 바로 임베디드 웹 서버의 산출물이다. 풀 스펙 아파치 대신 극도로 경량화된 서버가 칩 안에서 동작하고 있는 셈이다.
